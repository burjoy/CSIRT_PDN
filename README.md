
# How To Run



## Run Locally

Clone the project

```bash
  git clone https://github.com/burjoy/CSIRT_PDN/tree/main
```

Go to the project directory

```bash
  cd frontend
  cd server
```

Install dependencies

```bash
  npm install
```

Start the frontend

```bash
  npm run dev
```

Start the server
```bash
  npm start index.js
```
